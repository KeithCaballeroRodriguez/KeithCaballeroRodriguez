## 🚀 Proyecto Destacado

### [Mining Process Quality Prediction](https://github.com/KeithCaballeroRodriguez/minsur-case-study)

Sistema end-to-end de predicción de calidad para una planta de flotación de mineral de hierro. Predice **% Silica Concentrate** antes de que el laboratorio confirme el resultado (~1h de anticipación), permitiendo al operador actuar de forma proactiva.

| Componente | Detalle |
|------------|---------|
| **Modelo** | LightGBM + Optuna (60 trials, TimeSeriesSplit 5-fold) |
| **Métricas** | MAE ~0.47% · R² ~0.64 · Tol±20%: 62% |
| **Explainability** | SHAP global + waterfall local por predicción |
| **Agente IA** | Grafo LangGraph 10 nodos — predicción + SHAP + escenarios → recomendaciones |
| **Deploy** | FastAPI + Streamlit + MLflow en GCP Cloud Run |
